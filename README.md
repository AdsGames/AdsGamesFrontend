## Amplify

Installing

```sh
npm install -g @aws-amplify/cli
```

Pull (you must have access to the aws resources)

```sh
amplify pull
```

Link to your logged in aws profile (`aws configure`) and set the preset settings.

Pushing Changes

```sh
amplify push
```
